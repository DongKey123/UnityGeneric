# 스탯 설계

---

## 기본 스탯 목록

| 스탯 | 설명 | 영향 요소 |
|------|------|-----------|
| `MaxHp` | 최대 체력 | 투구 / 갑옷 / 능력치강화 |
| `Atk` | 공격력 | 무기 / 능력치강화 |
| `Def` | 방어력 | 갑옷 / 장갑 / 신발 / 능력치강화 |
| `AtkSpeed` | 공격 속도 | 장갑 / 능력치강화 |
| `AtkRange` | 공격 사거리 | 무기 고정값 |
| `MoveSpeed` | 이동 속도 | 신발 / 능력치강화 |
| `CritRate` | 치명타 확률 | 능력치강화 / 문신 |
| `CritDmg` | 치명타 피해 배율 | 능력치강화 / 문신 |
| `ExpBonus` | 경험치 획득 보너스 (%) | 능력치강화 / 문신 |
| `GoldBonus` | 골드 획득 보너스 (%) | 능력치강화 / 문신 |
| `ItemDropRate` | 아이템 드롭률 보너스 (%) | 능력치강화 / 문신 |
| `IdleGoldBonus` | 방치 골드 효율 보너스 (%) | 능력치강화 / 문신 |
| `IdleExpBonus` | 방치 경험치 효율 보너스 (%) | 능력치강화 / 문신 |

- **레벨은 스탯에 영향 없음** — 챕터 해금 조건으로만 사용
