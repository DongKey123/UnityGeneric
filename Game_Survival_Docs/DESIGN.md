# 서바이벌 게임 기획서

> Last Day on Earth: Survival 스타일의 탑다운 모바일 서바이벌 게임

---

## 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 탑다운 서바이벌 |
| 시점 | 쿼터뷰 |
| 배경 | 미정 |
| 플랫폼 | 모바일 (iOS / Android) |
| 핵심 루프 | 탐험 → 자원 수집 → 제작 → 베이스 강화 → 더 깊은 탐험 |

---

## 핵심 루프

```
베이스
  └─ 장비 / 아이템 준비
       └─ 탐험 지역 이동
            └─ 자원 수집 / 전투 / 루팅
                 └─ 베이스 귀환
                      └─ 제작 / 건설 / 업그레이드
                           └─ (반복)
```
