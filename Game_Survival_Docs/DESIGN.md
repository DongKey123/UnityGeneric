# 서바이벌 게임 기획서

> Last Day on Earth: Survival 스타일의 탑다운 모바일 서바이벌 게임
> 세계관 / 리소스 명칭은 미정 — 코딩 가능한 수준의 시스템 정의 중심으로 작성

---

## 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 탑다운 서바이벌 |
| 시점 | 쿼터뷰 |
| 배경 | **미정** |
| 플랫폼 | 모바일 (iOS / Android) |
| 핵심 루프 | 베이스 → 탐험 → 자원 수집 → 제작 → 베이스 강화 → 반복 |
| PvP (레이드) | **미정** |
| 장기 목표 | **미정** |

---

## 핵심 루프

```
베이스
  └─ 장비 / 아이템 준비
       └─ 탐험 지역 이동 (월드맵)
            └─ 자원 수집 / 적 처치 / 루팅
                 └─ 베이스 귀환
                      └─ 제작 / 건설 / 업그레이드
                           └─ (반복)
```

---

## 시스템별 최소 설계

### 1. 맵 / 이동

| 항목 | 결정 |
|------|------|
| 방식 | 지역 선택 이동 (오픈 월드 아님) |
| 구조 | 베이스 + 탐험 지역들 (씬 분리 or 단일 씬 — **미정**) |
| 지역 종류 | 베이스 / 일반 탐험 / 고위험 탐험 / 특수 (미정) |
| 지역 간 이동 | 월드맵 UI에서 선택 |

---

### 2. 플레이어

| 항목 | 결정 |
|------|------|
| 레벨 시스템 | **미정** (장비 품질로 성장 vs 레벨 혼용) |
| 기본 스탯 | HP / 스태미나 / 인벤토리 무게 한도 |
| 사망 처리 | 인벤토리 일부 드롭 (상세 비율 미정) |
| 장비 슬롯 | 무기 / 방어구 / 도구 (슬롯 수 미정) |

---

### 3. 인벤토리

| 항목 | 결정 |
|------|------|
| 방식 | 슬롯 + 무게 이중 제한 |
| 슬롯 수 | **미정** |
| 무게 단위 | **미정** |
| 스택 | 자원류 스택 가능 / 장비류 스택 불가 |
| 저장 | 베이스 내 상자에 보관 (인벤토리와 분리) |

---

### 4. 자원

> 명칭은 미정. 구현 시 ID 기반 테이블로 관리.

| 분류 | 예시 (임시) | 획득처 |
|------|------------|--------|
| 기본 재료 | 나무, 돌 | 채집 |
| 가공 재료 | 철, 천 | 채집 / 적 드롭 |
| 음식 | - | 사냥, 채집 |
| 희귀 부품 | - | 고위험 지역 루팅 |

---

### 5. 크래프팅

| 항목 | 결정 |
|------|------|
| 방식 | 레시피 테이블 기반 (JSON) |
| 작업대 | 단계별 (맨손 → 작업대 → 상위 작업대) |
| 해금 방식 | **미정** (레시피 자동 해금 vs 조건 해금) |

---

### 6. 베이스 건설

| 항목 | 결정 |
|------|------|
| 방식 | 그리드 기반 배치 |
| 건물 종류 | 바닥 / 벽 / 문 / 가구 (작업대, 상자) |
| 내구도 | 건물별 내구도 있음 |
| 재료 등급 | 등급에 따라 내구도 차이 (세부 미정) |

---

### 7. 전투

| 항목 | 결정 |
|------|------|
| 시점 | 탑다운 |
| 무기 종류 | 근접 / 원거리 |
| 스태미나 | 공격 / 달리기 시 소모 |
| 적 (Enemy) | 세계관 미정 — 코드상 `Enemy`로 통일 |
| 적 종류 | 일반 / 빠른 / 탱커 (명칭 미정) |
| 적 AI | 탐지 → 추격 → 공격 |
