# 베이스 건설 시스템 설계

---

## 개요

플레이어의 거점. 그리드 기반으로 건물을 배치하고 자원을 저장하며 제작을 진행.
베이스는 항상 접근 가능한 안전 지역.

---

## 건설 방식

| 항목 | 결정 |
|------|------|
| 방식 | 그리드 기반 배치 (타일 단위) |
| 그리드 크기 | **미정** |
| 배치 제한 | 기존 건물과 겹침 불가, 그리드 범위 내 |
| 철거 | 배치 후 철거 가능, 재료 일부 반환 |

---

## 건물 분류

### 구조물 (Structure)

| 건물 | 설명 |
|------|------|
| 바닥 | 건물 배치 기반, 없어도 배치 가능 (미정) |
| 벽 | 외부 침입 차단, 내구도 있음 |
| 문 | 출입구, 잠금 가능 |
| 지붕 | **미정** |

### 가구 / 기능물 (Furniture)

| 건물 | 설명 |
|------|------|
| 저장 상자 | 아이템 보관 |
| 작업대 (티어별) | 크래프팅 가능 범위 결정 |
| 캠프파이어 | 음식 조리 |
| 침대 | 체력 회복 / 오프라인 보상 거점 |

---

## 내구도 & 재료 등급

| 등급 | 예시 재료 (TBD) | 내구도 |
|------|----------------|--------|
| 1등급 | 나무 | 낮음 |
| 2등급 | 돌 | 중간 |
| 3등급 | 금속 | 높음 |

- 벽 / 문 내구도가 0이 되면 파괴
- 수리 가능 (재료 소모)

## 미결 사항

- [ ] 그리드 크기 (맵 대비 베이스 크기)
- [ ] 바닥 없이 벽 배치 가능 여부
- [ ] 지붕 시스템 포함 여부
- [ ] 베이스 확장 방식 (초기 범위 제한 여부)
- [ ] PvP 레이드 포함 시 침입 판정 상세
