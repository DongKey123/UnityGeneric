# Last Day on Earth: Survival — 게임 레퍼런스

> 이 문서는 우리가 만들 서바이벌 게임의 레퍼런스인 LDOE를 분석한 문서입니다.
> 설계 시 참고 자료로 활용하세요.

---

## 한 줄 요약

> 좀비 아포칼립스 세계에서 **자원을 수집하고 베이스를 짓고 살아남는** 탑다운 모바일 서바이벌

---

## 게임 흐름 (한 세션)

```
로그인
  └─ 베이스 확인 (오프라인 중 레이드 피해 여부)
       └─ 아이템 제작 / 장비 준비
            └─ 탐험 지역 이동 (월드맵)
                 └─ 자원 수집 / 좀비 처치 / 루팅
                      └─ 베이스 귀환 (사망 시 인벤토리 일부 손실)
                           └─ 자원 가공 / 건물 업그레이드
                                └─ 로그아웃 → 오프라인 타이머 진행
```

---

## 핵심 시스템

### 1. 월드맵 & 지역 구조

- 게임은 **오픈 월드가 아님** — 월드맵에서 지역을 선택해 이동하는 방식
- 지역 난이도 색상으로 구분
  - 초록 지역 — 일반 숲, 기본 자원 (나무, 돌)
  - 노란 지역 — 중간 위험, 좋은 자원
  - 빨간 지역 — 고위험, 희귀 자원 / 보스
  - 특수 지역 — 벙커, 군사 기지, 폐공장 등 (특정 아이템 필요)
- 각 지역은 **독립된 맵**으로 구성, 한 번 입장하면 탈출하거나 사망할 때까지 머뭄

---

### 2. 플레이어

- 레벨 시스템 없음 — **장비 품질**이 곧 성장
- 체력 (HP) / 배고픔 / 갈증 관리
- 사망 시 인벤토리 일부 또는 전체 드롭 (아이템 위험 부담)
- 장비 슬롯: 무기 / 방어구 (머리, 몸, 다리) / 도구

---

### 3. 자원 수집

| 자원 | 획득처 |
|------|--------|
| 나무 | 나무 벌목 |
| 돌 / 철 | 광석 채굴 |
| 음식 | 동물 사냥, 채집 |
| 천, 가죽 | 좀비 처치, 동물 |
| 희귀 부품 | 특수 지역 루팅 |

- 도끼 / 곡괭이 등 **도구**가 있어야 수집 가능 (맨손 채집은 비효율)
- 인벤토리 **무게 제한** 있음 → 한 번에 많이 못 가져옴

---

### 4. 인벤토리

- 슬롯 + 무게 두 가지 제한
- 장비 슬롯 (착용), 일반 슬롯 (소지)
- 베이스의 **저장 상자**에 보관 → 인벤토리와 분리

---

### 5. 크래프팅 (제작)

- 레시피 기반 — 재료 + 작업대 조합
- **작업대 종류**에 따라 만들 수 있는 것이 다름

| 작업대 | 제작 가능 |
|--------|-----------|
| 맨손 | 기본 도구, 임시 무기 |
| 캠프파이어 | 음식 조리 |
| 작업대 | 무기, 방어구, 건축 재료 |
| 총기 작업대 | 총기, 탄약 |

- 더 좋은 작업대를 만들려면 더 좋은 재료가 필요 → **점진적 성장 구조**

---

### 6. 베이스 건설

- **그리드 기반** 건설 (타일 단위 배치)
- 건물 종류: 바닥, 벽, 문, 계단, 지붕
- 재료에 따라 내구도 차이 (나무 < 돌 < 금속)
- 벽과 문에 **내구도** 있음 → 레이드 시 부숴짐
- 상자, 침대, 작업대 등 **가구** 배치

---

### 7. 전투

- 탑다운 근접 / 원거리 전투
- 좀비 종류별 특성
  - 일반 좀비 — 느리고 약함
  - 달리는 좀비 — 빠름
  - 탱커 좀비 — 느리고 체력 높음
  - 특수 좀비 — 특정 지역 전용
- 스태미나 시스템 — 달리기, 공격 시 소모

---

### 8. 레이드 (PvP)

- 오프라인 중 다른 플레이어가 내 베이스를 공격 가능
- 벽 / 문 내구도를 0으로 만들면 침입
- 베이스 방어 설계가 중요 (미로 구조, 강화 문 등)

---

### 9. 오프라인 보상 & 타이머

- 오프라인 중에도 **베이스 내 자원 생산** (발전기, 농장 등)
- 일정 시간마다 좀비 무리가 베이스 공격 (주기적 레이드)
- 재접속 시 오프라인 시간만큼 보상 수령

---

## 핵심 장기 목표

LDOE의 최종 목표는 **오토바이(초퍼) 제작** — 희귀 부품을 모아 탈것을 완성하는 장기 목표가 플레이어를 계속 접속하게 만드는 동기.

---

## 우리 게임에서 참고할 포인트

| 시스템 | 참고 방향 |
|--------|-----------|
| 월드맵 → 지역 이동 | 오픈 월드 대신 지역 선택 방식 → 개발 부담 감소 |
| 무게 제한 인벤토리 | 자원 관리 긴장감 |
| 작업대 단계별 제작 | 점진적 성장 구조 |
| 그리드 베이스 건설 | 건설 시스템의 핵심 |
| 오프라인 보상 | 모바일 재접속 유도 |
| 장기 목표 (초퍼) | 플레이어 리텐션 핵심 |
