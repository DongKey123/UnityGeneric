# 전투 시스템 설계

---

## 개요

탑다운 뷰에서 근접 / 원거리 무기로 적(Enemy)을 처치.
스태미나 관리가 전투 리듬을 결정.

---

## 플레이어 전투

| 항목 | 결정 |
|------|------|
| 시점 | 탑다운 |
| 공격 방식 | 무기 장착 후 공격 버튼 or 적 터치 (미정) |
| 무기 종류 | 근접 / 원거리 |
| 스태미나 | 공격 / 달리기 시 소모, 시간 경과로 회복 |
| 회피 | **미정** |
| 피격 | HP 감소 + 경직 효과 (미정) |

### 무기 분류

| 종류 | 특성 |
|------|------|
| 근접 | 짧은 사거리, 빠른 공격속도, 스태미나 소모 낮음 |
| 원거리 | 긴 사거리, 탄약 소모, 스태미나 소모 없음 |
| 도구 (근접 겸용) | 채집 도구로도 사용 가능 |

---

## 적 (Enemy)

> 세계관 미정 — 코드상 `Enemy`로 통일, 명칭은 추후 변경

### 적 종류

| 타입 | 특성 |
|------|------|
| 기본형 | 느리고 약함, 근접 공격 |
| 고속형 | 빠름, 약한 공격 |
| 탱커형 | 느리고 HP 높음, 강한 공격 |
| 특수형 | 지역별 전용 (상세 미정) |

### 적 AI

```
Idle (배회)
  └─ 탐지 범위 내 플레이어 감지
       └─ Chase (추격)
            └─ 공격 사거리 진입
                 └─ Attack (공격)
                      └─ 플레이어 사거리 이탈 → Chase 복귀
                           └─ 사망 → Dead (드롭 후 제거)
```

| 항목 | 결정 |
|------|------|
| 탐지 방식 | 시야 반경 (테이블로 적별 관리) |
| 군집 행동 | **미정** |
| 피격 반응 | 피격 시 추격 시작 (시야 밖이어도) |

---

## 데미지 계산

```
최종 데미지 = max(공격력 × 무기배율 × 크리티컬 - 방어력 × 0.5, 최솟값)
```

> 수치는 미정, 공식은 Idle RPG와 동일 구조 재사용 가능

---

## 사망 처리

| 대상 | 처리 |
|------|------|
| 적 사망 | 자원 / 아이템 드롭, 일정 시간 후 제거 |
| 플레이어 사망 | 인벤토리 일부 드롭, 베이스로 귀환 |

---

## 미결 사항

- [ ] 공격 조작 방식 (버튼 vs 터치 타겟)
- [ ] 회피 시스템 포함 여부
- [ ] 군집 행동 구현 여부
- [ ] 플레이어 사망 시 드롭 비율
- [ ] 크리티컬 시스템 포함 여부
- [ ] 적 명칭 (세계관 확정 후)
