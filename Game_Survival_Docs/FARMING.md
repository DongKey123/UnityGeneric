# 파밍 (자원 수집) 시스템 설계

---

## 개요

탐험 지역에 배치된 자원 오브젝트를 터치/상호작용으로 수집.
수집한 자원은 인벤토리에 저장되며, 베이스로 가져가 제작에 사용.

---

| 분류 | 임시 명칭 | 획득처 | 도구 필요 |
|------|-----------|--------|-----------|
| 기본 재료 A | 나무 (TBD) | 나무 오브젝트 | 도끼류 |
| 기본 재료 B | 돌 (TBD) | 바위 오브젝트 | 곡괭이류 |
| 가공 재료 A | 철 (TBD) | 광석 오브젝트 | 곡괭이류 |

---

## 수집 방식

- 자원 오브젝트에 **근접 후 터치** → 수집 시작
- 수집 시간 (채집 딜레이) — **도구 티어에 따라 단축**
- 맨손 수집 가능하나 비효율 (시간 오래 걸림, 일부 불가)
- 수집 중 적에게 피격 시 **중단**

---

## 자원 오브젝트

| 항목 | 내용 |
|------|------|
| 배치 방식 | 지역 입장 시 랜덤 배치 (밀도는 테이블로 관리) |
| 내구도 | 오브젝트마다 수집 가능 횟수 존재 → 0이 되면 제거 |
| 리스폰 | 지역 재입장 시 다시 생성 (일정 시간 후 or 항상) — **미정** |
| 드롭량 | 테이블 기반 (min ~ max 랜덤) |

---

## 도구 시스템

- 도구 없이도 수집 가능하나 효율 낮음
- 도구 티어에 따라 수집 속도 / 드롭량 증가
- 도구도 **내구도** 있음 → 사용 시 마모, 0이 되면 파괴

| 도구 | 대상 자원 |
|------|-----------|
| 도끼류 | 나무 계열 |
| 곡괭이류 | 돌 / 광석 계열 |
| 낫류 | 식물 계열 |

---

## 미결 사항

- [ ] 자원 명칭 (세계관 확정 후)
- [ ] 지역별 자원 밀도 테이블
- [ ] 리스폰 방식 (재입장 시 항상 vs 일정 시간 쿨타임)
- [ ] 도구 내구도 수치
- [ ] 드롭량 min/max 테이블
