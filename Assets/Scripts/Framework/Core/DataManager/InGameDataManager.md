# InGameDataManager

JSON 파일을 로드하고 타입 안전하게 캐싱하는 런타임 게임 데이터 매니저입니다.

---

## 개요

ExcelToJson으로 변환된 JSON 파일을 Runtime에 로드하여 Dictionary 또는 List 형태로 캐싱합니다.
중첩 제네릭 정적 클래스 구조로 boxing 없이 타입별 독립 캐시를 유지하며,
`ExcelClassGenerator`의 InGameDataManager 연동 기능으로 시트별 타입 안전 Get 메서드를 자동 생성할 수 있습니다.

---

## 의존성

- `Framework.Core.Singleton` — MonoBehaviour 싱글톤 베이스
- [Newtonsoft.Json](https://www.newtonsoft.com/json) — JSON 역직렬화
- `DataParser/ExcelToJson` — JSON 파일 생성 (선택, 연동 시)

---

## API 레퍼런스

### 클래스

| 이름 | 설명 |
|------|------|
| `InGameDataManager` | JSON 로드 + 캐싱 런타임 매니저 (partial, Singleton) |

### 주요 메서드

| 메서드 | 반환 타입 | 설명 |
|--------|-----------|------|
| `Load<T>(resourcePath)` | `List<T>` | JSON을 List로 로드. 캐시 있으면 캐시 반환 |
| `LoadAsDictionary<TKey, TValue>(resourcePath, keySelector)` | `Dictionary<TKey, TValue>` | JSON을 Dictionary로 로드. 캐시 있으면 캐시 반환 |
| `Unload<T>(resourcePath)` | `void` | List 캐시 해제 |
| `UnloadDictionary<TKey, TValue>(resourcePath)` | `void` | Dictionary 캐시 해제 |
| `UnloadAll()` | `void` | 모든 캐시 해제 |

---

## 사용법

### 기본 사용

```csharp
// List로 로드
var monsters = InGameDataManager.Instance.Load<MonsterData>("Data/Monster");

// Dictionary로 로드 (id 기준 빠른 조회)
var monsterDict = InGameDataManager.Instance.LoadAsDictionary<int, MonsterData>(
    "Data/Monster",
    x => x.id
);

// 단건 조회
if (monsterDict.TryGetValue(101, out var monster))
    Debug.Log(monster.name);
```

### 캐시 해제

```csharp
// 특정 경로 해제
InGameDataManager.Instance.Unload<MonsterData>("Data/Monster");
InGameDataManager.Instance.UnloadDictionary<int, MonsterData>("Data/Monster");

// 전체 해제 (씬 전환 등)
InGameDataManager.Instance.UnloadAll();
```

## 예시 (심화)

### ExcelClassGenerator 연동

`DataParser/ExcelToJson`의 **C# 클래스 생성 + InGameDataManager 연동** 기능을 활성화하면
시트별 Get 메서드가 자동 생성됩니다.

```csharp
// 자동 생성된 partial 메서드 사용
var monsters = InGameDataManager.Instance.GetMonster();  // Dictionary<int, MonsterData>
var items    = InGameDataManager.Instance.GetItem();     // Dictionary<int, ItemData>
```

자동 생성 파일 예시 (`InGameDataManager.Monster.Generated.cs`):

```csharp
// Auto-generated by ExcelToJson. Do not edit manually.
using System.Collections.Generic;
using GameData;

namespace Framework.Core.DataManager
{
    public partial class InGameDataManager
    {
        public Dictionary<int, MonsterData> GetMonster()
            => LoadAsDictionary<int, MonsterData>("Data/Monster", x => x.id);
    }
}
```

## 주의사항

- `Resources` 폴더 기반으로 동작합니다. `resourcePath`는 `Assets/Resources/` 이후 경로, 확장자 제외로 작성합니다.
  - 예) `"Data/Monster"` → `Assets/Resources/Data/Monster.json`
- 씬 전환 시 `UnloadAll()`을 호출해 메모리를 정리하세요.
- 자동 생성된 partial 클래스 파일(`// Auto-generated`)은 직접 수정하지 마세요.
- 플레이 모드 재시작 시 `[RuntimeInitializeOnLoadMethod]`로 모든 캐시가 자동 초기화됩니다.
